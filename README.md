# World Higher Education Database (WHED)

The **World Higher Education Database (WHED)** is a comprehensive source of information on higher education systems and institutions worldwide. This project is developed in partnership with the **International Association of Universities (IAU)** and **UNESCO**.

## 🚀 Technologies Used

- **Framework:** [Next.js 16](https://nextjs.org/) (React 19)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/)
- **State Management:** [Zustand](https://github.com/pmndrs/zustand)
- **Icons:** [React Icons](https://react-icons.github.io/react-icons/)
- **Font:** Geist & Geist Mono

## 📂 Project Structure

```text
src/
├── app/               # Next.js App Router (Layouts, Pages, Styles)
├── components/        # Reusable UI Components
│   └── shared/        # Shared components like Header and Footer
└── stores/            # State management with Zustand
public/                # Static assets
```

## 🛠️ Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed (v18.17 or later recommended).

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd whed
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📜 Scripts

- `npm run dev` - Starts the development server.
- `npm run build` - Builds the application for production.
- `npm run start` - Starts the production server.
